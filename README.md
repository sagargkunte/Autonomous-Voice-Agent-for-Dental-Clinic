# Autonomous Voice Agent for Dental Clinic

An intelligent voice-based assistant system designed to streamline dental clinic operations through automated appointment scheduling, patient inquiries, and administrative support.

## Overview

This project implements an autonomous voice agent that leverages natural language processing and voice recognition to handle common dental clinic operations. The system is built with modern web technologies to provide a seamless user experience.

## Features

- **Voice Recognition**: Convert spoken language into actionable commands
- **Appointment Scheduling**: Automated booking and management of dental appointments
- **Patient Information Management**: Store and retrieve patient details efficiently
- **Inquiry Handling**: Respond to common patient questions about procedures and services
- **Multi-language Support**: Communicate with patients in their preferred language
- **Integration Ready**: Easy integration with existing dental clinic management systems

## Tech Stack

- **Frontend**: JavaScript (97.8%)
- **Styling**: CSS (2.1%)
- **Markup**: HTML (0.1%)

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn package manager
- Modern web browser with Web Audio API support

### Installation

1. Clone the repository:
```bash
git clone https://github.com/sagargkunte/Autonomous-Voice-Agent-for-Dental-Clinic.git
cd Autonomous-Voice-Agent-for-Dental-Clinic
```

2. Install dependencies:
```bash
npm install
```

3. Configure environment variables:
```bash
cp .env.example .env
# Edit .env with your configuration
```

4. Start the application:
```bash
npm start
```

The application will be available at `http://localhost:3000` (or your configured port).
